DigiLocker

Your document anywhere, anytime.

The All-in-One documents care app which provide your docs anywhere, anytime.

Over view

Digilocker is an application to store your documents and certificates. Which helps in avoiding the need of carrying them physically.

The documents are electronically generated in Digilocker and verified by issued organizations/authorities. This application allows you to access your documents anywhere at any time.

My Task

The task was to identify the issues in the existing application and improve the features.

My research, findings and suggestions

Digilocker is a well-thought application that fulfills its purpose as promised.

Now when we talk about user experience for this application, there can be few areas where a slight improvement or new feature might help in smoothing the whole process for the users. Few of them -

  • Language Switcher
  • Consistant Layout
  • Finger-print login & Sessions
  • Reporting Issue & knowledge base
  • Notification & retention of information
  • Document Scanner

Language Switcher

As we all know, India is a vast country with many languages.
Introducing new languages in the application can increase the userbase many times.

Consistant Layout

A product should follow the consistency from onboarding to exiting.
The layout of bottom navigation changes when the user login or register to his/her account.

Finger-print Logins & Sessions

The security of these sensitive documents is a must.
